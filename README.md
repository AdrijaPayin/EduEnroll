# Student Application Form

Live demo : https://adrijapayin.github.io/student-application-form/

This project is a simple web-based Student Application Form built using HTML, CSS, and basic JavaScript. The form collects student information, academic details, and ensures proper validation before submission.

### Features
* Input fields for student personal details
* Class selection using radio buttons
* Marks entry table with validation (0–100)
* Phone number validation (10 digits)
* PIN code validation (6 digits)
* Only one class selectable
* Terms & Conditions must be accepted before submission
* Submit button activates only after all conditions are checked

### Technologies Used
HTML5
CSS
JavaScript (Form Validation)
